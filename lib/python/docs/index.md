# QuestFoundry Python Library

Welcome to the QuestFoundry Python library documentation!

## Overview

QuestFoundry is a Python library for collaborative interactive fiction authoring. It provides:

- **Schema Validation**: JSON schema validation for QuestFoundry artifacts
- **Prompt Management**: Access to AI agent prompts for various roles
- **Resource Bundling**: Bundled schemas and prompts from the specification
- **Provider Integration**: Support for multiple LLM providers (OpenAI, Ollama, Google, etc.)

## Key Features

- 🎭 **Role-Based Architecture**: 15 specialized studio roles for narrative production
- 🔄 **Loop-Focused Design**: 12 production loops for structured workflows
- 📋 **Artifact Management**: 30+ schema-validated artifact types
- 🤖 **AI Agent Integration**: Pre-configured prompts for AI agents
- 🔒 **Type Safety**: Full type annotations and mypy support
- ✅ **Validation**: Comprehensive schema validation with jsonschema

## Project Structure

This library is part of the QuestFoundry mono-repo:

- **Specification** (`spec/`) - Schemas, prompts, and documentation
- **Libraries** (`lib/python/`) - This Python implementation
- **CLI Tools** (`cli/`) - Command-line interface tools
