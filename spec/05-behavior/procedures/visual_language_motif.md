# Visual Language & Motif

## Purpose

Establish and maintain a consistent visual motif palette that aligns with the project's register and enhances narrative coherence across all illustrations.

## Core Principles

- **Style Alignment**: Visual language must harmonize with Style Lead's register, voice, and tone
- **Motif Consistency**: Recurring visual elements should be recognizable and meaningful
- **Narrative Support**: Motifs should reinforce themes without telegraphing spoilers
- **Cross-Surface Coherence**: Visual language applies to all illustration contexts (manuscript, codex, UI)

## Steps

1. **Coordinate with Style Lead**: Review register map and motif kit
2. **Define Visual Palette**: Establish color schemes, iconographic patterns, compositional styles
3. **Document Motif Use**: Record which motifs apply to which narrative contexts
4. **Review Existing Plans**: Audit art plans for motif consistency
5. **Provide Illustrator Guidance**: Share motif palette and application notes
6. **Monitor Consistency**: Review rendered work for adherence to visual language
7. **Evolve Thoughtfully**: Update motifs only with Style Lead coordination

## Outputs

- **Visual Motif Palette**: Documented set of recurring visual elements
- **Application Guidelines**: Notes on when and how to use specific motifs
- **Style Coordination Notes**: Alignment records with Style Lead
- **Consistency Audits**: Reviews of illustration work for motif adherence

## Quality Checks

- Motif palette aligns with Style Lead's register and voice
- Visual elements consistently applied across all illustrations
- Motifs enhance narrative without revealing spoilers
- Changes to motif palette are coordinated with Style Lead
- Documentation enables consistent application by Illustrator
