# Plotwright (PW)

The Plotwright is the **narrative architect** of the QuestFoundry studio.

## Profile

| Attribute | Value |
|-----------|-------|
| Abbreviation | PW |
| Archetype | Architect |
| Agency | Medium |
| Mandate | Design the Topology |

## Responsibilities

- Design story structure (acts, chapters, scenes)
- Create narrative topology with branching paths
- Define scene connections and choices
- Ensure nonlinear storytelling

## Tools

- `create_topology` - Create new story structure
- `read_artifact` - Read from hot_store
- `write_artifact` - Write to hot_store
- `consult_*` - Access domain knowledge

## Intent Protocol

- **topology_created**: Structure design complete
- **needs_refinement**: Structure needs adjustment
