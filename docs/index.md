# Welcome to QuestFoundry

**QuestFoundry** is a layered architecture for collaborative interactive fiction authoring. This mono-repo contains the complete project, from specification to implementation.

This site contains the unified documentation for all components:

- **[Specification (L0-L5)](spec/README.md):** The canonical definition of QuestFoundry, including roles, artifacts, schemas, and AI prompts.
- **[Python Library (L6)](lib/python/README.md):** The Layer 6 runtime implementation for state, protocol, and AI orchestration.
- **[CLI Tools (L7)](cli/python/README.md):** (Planned) Command-line tools for interacting with the QuestFoundry ecosystem.
- **[Schema Definitions](schemas/index.md):** The publicly hosted JSON schemas, required for validation and client tooling.
